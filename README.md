### ğŒ How to apply the GlassScripts to your JellyFin? 

<b>Use the following information to install JavaScript Injector</b><br>
_Require to make GlassScripts work_<br>
[How to install Jellyfin JavaScript Injector](https://github.com/n00bcodr/Jellyfin-JavaScript-Injector?tab=readme-ov-file#%EF%B8%8F-installation)

<b>Follow steps once Jellyfin JavaScript Injector is installed</b>
<details>
  <summary><i>JellyFin Administrator/Server side | In JellyFin Version 10.11.x</i></summary>

1. Open the Dashboard option from the side mega menu.
2. From the side menu, select the JS Injector tab within the Plugins area
3. Select Add Script
4. Create a name for the script you wish to use
5. Copy the raw code of the script and paste it into the textarea
6. By standard, make sure Enabled is ticked and dependant on the script, select Requires Authentication if you do not want it displaying on the login page
7. Click save
</details>
